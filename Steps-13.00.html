Building Datawarehouse in SSIS 13

This project involves setting up a Staging area, ODS, and Data Warehouse, starting from JSON data. Below is a screenshot illustrating the data warehouse model and the ETL solution package we’ll be developing in this project.

Data Warehouse Data Model:



ETL SOLUTION PACKAGE:


Staging Layer: This is where raw data from multiple sources is collected and stored with minimal or no transformation.

ODS Layer: The Operational Data Store (ODS) contains near real-time data, as defined by organizational needs. In this layer, data is transformed, validated, and business logic is applied through ETL processes.

Data Warehouse: This is where transformed data from the ODS is stored, holding historical information for long-term analysis.

The tools used for this project include:

SSMS – Utilized for executing SQL scripts.

SSIS (SQL Server Integration Services) - An ETL tool used for data transformation, validation, and loading SQL tables.
