3. ETL SOLUTION FOR DATA WAREHOUSE LAYER:

  • Add Sequence Container for DWH Layer
  • Then add two Execute SQL Task to load DIM and FACT tables into Sequence Container and connect them.
  • Name the task as Load Dim Tables and Load Fact Tables.


               


  • Inside Load Dim Tables Execute SQL Task in SQLStatement section and the query to execute the Store Procedure to load Dim Tables.


         


  • Inside Load Fact Tables Execute SQL Task in SQLStatement section and the query to execute the Store Procedure to load Fact Tables.

          


  • Load Staging, ODS, DWH. Execute the package.


          


  • After package completes stop the package and check whether data loaded successfully.
