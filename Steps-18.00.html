4. Inside the Data Flow Task (Dictionary Tables)


  • Table 'tbl_Dictionary_Location'

             



  • Inside the OLEDB Source configure the staging table connection and use SQL command to get the data.

         

          



  • In OLEDB Source Editor, use column tab to map all the input to output columns


            



  • Use Derived Column Transformation to generate InsertedDate and InsertedBy Data.

 

       



  •  LOOKUP TRANSFORMATION
      Select rows with no matching entries.


          



  • Use Lookup Connection tab to specify the data source.


          



  • Use Lookup Columns tab to enable joins between the inputs and reference data set.


        



  • In OLEDB Destination create connection to connect with tbl_Dictionary_Location table.


          



  • Use tbl_Dictionary_Location Mapping tab to map input output columns.


         



  • Use OLEDB COMMAND TRANSFORMATION to update the Dictionary_Location records.


           



  • In Component properties use SqlCommand section to write the update statement.


           



  • Use Column Mappings tab to map input and output columns


          



  • Repeat the same exact steps for other Dictionary tables as above designed for tbl_Dictionary_Location.
